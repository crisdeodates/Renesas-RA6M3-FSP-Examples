1. Project Overview:
        The example project the capability of the EK and FSP by usign multiple preipherals.
        The project initializes the USB,ADC and GPIO modules with control via a USB Comm Device class termial.
        Once programmed the user can control the board by button presses on the board and see regulaly updated board 
        configuration on the kit information screen.

2. Hardware settings for the project:
        Please configure the Jumper settings as described in the Quick Start Guide.
	Link to access Quick Start Guide - https://www.renesas.com/us/en/document/qsg/ek-ra4m2-quick-start-guide

3. Configuration Settings:
        Please explore the configuration settings of each module in the RA configurator.
