1. Project Overview:
        In the Quick Start Example Project, the user can change the control (rotation start/ stop, rotation speed change)
        of the permanent magnet synchronous motor, "TG-55L-KA" by TSUKASA Electric Co., Ltd,
        by combining with the inverter board supplied with "Motor Control Evaluation System for RA Family - RA6T1 Group".

2. Hardware settings for the project:
        Please configure the RA6T1 CPU Card and Inverter Board settings as described in the Quick Start Guide.

3. Configuration Settings:
        Please explore the configuration settings of each module in the RA configurator.
